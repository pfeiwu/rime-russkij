# russkij: 基於 Rime 實現的 俄語 + 其他斯拉夫語 輸入方案
本方案旨在讓用戶在標準 QWERTY 鍵盤（非俄語鍵盤）上，高效輸入俄語及其他斯拉夫語言。所有編碼僅映射到 **26 個字母鍵** 及 **; '** 兩個符號鍵上。
## 特點
- **覆蓋全面**：支持所有斯拉夫語言所需的西里爾字母（Cyrillic）。
- **緊湊佈局**：僅佔用 28 個按鍵，比標準俄語鍵盤鍵位更集中，減少手指移動，更符合人體工學。
- **直覺易學**：編碼邏輯遵循發音規則，本質上是一套直觀的西里爾-拉丁轉寫系統（Transliteration）。懂發音即會打字，無需死記硬背鍵位。
- **無重碼設計**：編碼符合「前綴碼」特性，輸入路徑唯一，打字過程中無需使用數字鍵選詞，實現流暢盲打。
